# Storage Paths

- Config: `~/.smoovtask/config.toml`
- Events: `~/.smoovtask/events/YYYY-MM-DD.jsonl` (configurable via `settings.events_path`)
- Rules: `~/.smoovtask/rules/*.yaml` (seeded by `st install`)
- Tickets: `<vault_path>/tickets/YYYY-MM-DDTHH:MM-st_xxxxxx.md` (default vault: `~/obsidian/smoovtask`)
